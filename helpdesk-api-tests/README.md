# Testes Automatizados - Helpdesk API

## Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/helpdesk-api-tests.git
cd helpdesk-api-tests
```

2. Instale as dependências:
```bash
npm install
```

3. Execute os testes:
```bash
npx cypress open
```
